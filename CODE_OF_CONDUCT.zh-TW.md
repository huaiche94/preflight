# 貢獻者公約行為準則（Contributor Covenant Code of Conduct）

> 🌐 [English](CODE_OF_CONDUCT.md) | 繁體中文
>
> 本文件為非規範翻譯，內容以英文版為準（ADR-049）。

## 我們的承諾（Our Pledge）

身為社群的成員、貢獻者與領導者，我們承諾讓每一個人參與我們的社群時，都能享有免於騷擾的體驗，無論年齡、體型、可見或不可見的身心障礙、族裔、性徵、性別認同與表達、經驗程度、教育程度、社經地位、國籍、外貌、種族、宗教，或性傾向與性認同為何。

我們承諾以有助於打造開放、友善、多元、包容且健康社群的方式行事與互動。

## 我們的標準（Our Standards）

有助於為我們的社群營造正向環境的行為範例包括：

* 對他人展現同理心與善意
* 尊重不同的意見、觀點與經驗
* 給予並優雅地接受建設性回饋
* 為自己的錯誤承擔責任、向受影響的人致歉，並從中學習
* 不僅著眼於個人利益，而是著眼於整個社群的最大利益

不可接受的行為範例包括：

* 使用具性意味的語言或圖像，以及任何形式的性關注或性挑逗
* 酸言酸語（trolling）、侮辱或貶損性言論，以及人身或政治攻擊
* 公開或私下的騷擾
* 未經他人明確許可，公開他人的私人資訊，例如實體地址或電子郵件地址
* 在專業場合中可被合理認定為不恰當的其他行為

## 執行責任（Enforcement Responsibilities）

社群領導者負責釐清並執行我們對可接受行為的標準，並會針對他們認定為不恰當、具威脅性、冒犯性或有害的任何行為，採取適當且公平的糾正措施。

社群領導者有權利也有責任移除、編輯或拒絕不符合本行為準則的評論、commit、程式碼、wiki 編輯、issue 及其他貢獻，並會在適當時說明其審核決定的理由。

## 適用範圍（Scope）

本行為準則適用於所有社群空間，也適用於個人在公開場合正式代表本社群的情形。代表本社群的範例包括使用官方電子郵件地址、透過官方社群媒體帳號發文，或在線上或實體活動中擔任被指派的代表。

## 執行（Enforcement）

如遇辱罵、騷擾或其他不可接受的行為，可透過本 repository 私下的 **GitHub Security Advisory**
（Security 分頁 →「Report a vulnerability」——與
[`SECURITY.md`](SECURITY.md) 所述的私下回報管道相同；若沒有另外提供的行為準則專用回報信箱，也請使用這個管道回報行為問題）向負責執行的社群領導者回報。所有申訴都會被迅速且公正地審閱與調查。

所有社群領導者都有義務尊重回報者的隱私與安全。

## 執行準則（Enforcement Guidelines）

社群領導者在判斷任何被認定違反本行為準則之行為的後果時，將遵循以下社群影響準則（Community Impact Guidelines）：

### 1. 更正（Correction）

**社群影響**：使用不恰當的語言，或社群中認定為不專業或不受歡迎的其他行為。

**後果**：由社群領導者發出私下的書面警告，說明違規的性質，並解釋該行為為何不恰當。可能會要求公開道歉。

### 2. 警告（Warning）

**社群影響**：透過單一事件或一系列行為所構成的違規。

**後果**：警告，並附帶對持續行為的後果。在指定期間內不得與相關人士互動，包括對執行行為準則者的主動接觸。這也包括避免在社群空間及外部管道（如社群媒體）中的互動。違反這些條款可能導致暫時或永久禁止。

### 3. 暫時禁止（Temporary Ban）

**社群影響**：嚴重違反社群標準，包括持續的不恰當行為。

**後果**：在指定期間內暫時禁止與社群進行任何形式的互動或公開溝通。在此期間內不得與相關人士有公開或私下的互動，包括對執行行為準則者的主動接觸。違反這些條款可能導致永久禁止。

### 4. 永久禁止（Permanent Ban）

**社群影響**：展現出違反社群標準的持續模式，包括持續的不恰當行為、騷擾特定個人，或對特定族群的攻擊或貶損。

**後果**：永久禁止在社群內進行任何形式的公開互動。
